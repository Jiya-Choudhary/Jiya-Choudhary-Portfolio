## 🚀 Features

- **Modern Responsive UI** – Clean, modern, and minimal user interface built with Tailwind CSS.
- **Dark/Light Mode** – Seamless toggle between light and dark themes using `next-themes`.
- **Animated Interface** – Smooth animations and micro-interactions built with Framer Motion (Motion).
- **Projects Showcase** – Interactive directory displaying web applications, frontend UIs, and Python-based projects.
- **Skills Section** – Categorized display of languages, frameworks, databases, libraries, hard skills, and soft skills.
- **Achievements & Experience** – Highlights of certifications, academic honors, hackathons, and leadership roles.
- **GitHub Contributions** – Real-time visualization of GitHub activity and contributions.
- **Contact Integration** – Convenient links to connect via email, LinkedIn, and developer platforms.
- **Resume Integration** – Direct link to my resume within the navigation interface.
- **SEO Optimization** – Configured meta tags, keywords, and description for improved search visibility.
- **Mobile Responsive Design** – Fully optimized layouts for all screen sizes and mobile browsers.
- **Fast Performance** – Built on Next.js 16 and optimized with modern Geist fonts for high performance.

## ⚙️ Installation

To set up and run this portfolio project locally, follow these steps:

### Prerequisites

Make sure you have Node.js and `pnpm` installed on your machine.

### Setup Steps

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Jiya-Choudhary/Portfolio-Jiya-Choudhary.git
   cd Portfolio-Jiya-Choudhary
   ```

2. **Install Dependencies:**
   ```bash
   pnpm install
   ```

3. **Run the Development Server:**
   ```bash
   pnpm dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.
